## 안녕하세요, 잠탱이입니다

Java와 C#을 주로 다뤄온 현직 개발자입니다. 스마트팩토리 분야에서 현장 개발 업무를 하고 있어요.

이 블로그는 제가 AI 서비스 개발자로 전환하는 과정을 날것 그대로 기록하는 공간입니다. 완성된 이야기가 아니라, 지금 진행 중인 이야기예요.

---

## 왜 전환을 결심했나

솔직히 말하면, 위기감이었어요.

AI 없는 서비스가 없어지고 있고, 개발자로서 AI를 모르고선 점점 경쟁력이 떨어질 것 같다는 느낌이 들었어요. 트렌드라고 넘기기엔 변화의 속도가 너무 빠르더라고요.

그래서 목표를 잡았습니다.

> **6개월 안에 AI 서비스를 직접 만들 수 있는 개발자가 되자.**

이미 개발 베이스가 있으니까, 생각보다 현실적인 목표라고 믿고 있어요.

---

## 왜 Python인가

Java, C# 개발자로서 Python은 낯선 언어였어요.

근데 AI 생태계를 보면 선택지가 없어요. LangChain, FastAPI, PyTorch, Hugging Face — 전부 Python이에요. AI 개발을 하겠다면 Python이 사실상 표준이에요.

이번 기회에 **Python을 AI 개발에 맞는 방식으로** 제대로 익히는 것부터 시작하려고 해요.

---

## 기술 스택 목표

```
LLM API  : Gemini API (Google)
Backend  : FastAPI + uvicorn
Frontend : Vanilla HTML/CSS/JS
Vector DB: ChromaDB
Pipeline : LangChain
Database : SQLite
```
